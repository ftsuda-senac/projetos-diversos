Exemplo Spring + JPA

Objetivos

  • Explorar a aplicação (tanto código-fonte quanto através de navegação) para ver as funcionalidades e o comportamento delas
  • Alterar a aplicação para usar o JPA do JEE com banco de dados H2
  • Alterar a aplicação para usar o Spring Data JPA com banco de dados H2
  • Incluir funcionalidade para upload de fotos

Informações importantes

  • Este exemplo inicialmente usa alguns dados mockeados para possibilitar a navegação entre telas simulando a aplicação final
  • Conceitos já vistos anteriormente usados neste exemplo:
    • Criação de telas usando tecnologias web
      • HTML
      • CSS
      • Javascript
    • Criação de funcionalidades usando Spring MVC (GET/POST) + Thymeleaf
    • POST-Redirect-GET
    • Validação de dados
    • Componentização de seções das telas usando Layout Dialect
    • Criação de rota para acessar arquivos em diretórios externos
